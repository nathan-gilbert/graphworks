# Examples

Runnable scripts that demonstrate graphworks features. These are **not** shipped with the library
(the `examples/` directory is excluded from both the sdist and wheel).

## Running

The recommended way is via the `uv run` script alias defined in `pyproject.toml`:

```sh
uv run demo
```

You can also invoke the file directly:

```sh
uv run python examples/demo.py
```
